# h40y.github.io

A dark, minimal static blog for GitHub Pages.

## Structure

- `index.html` - homepage and post index
- `posts/` - individual HTML posts
- `styles.css` - shared dark-only visual style
- `feed.xml` - RSS feed
- `sitemap.xml` - search engine sitemap

## Writing

Copy an existing file in `posts/`, update the title/date/content, then add the post to `index.html`, `feed.xml`, and `sitemap.xml`.
